🐾 Adopta & Ama
Proyecto web desarrollado únicamente con HTML y CSS como parte de una materia académica.
📌 Descripción
Adopta & Ama es una página web estática que promueve la adopción de mascotas. Presenta información sobre animales disponibles, secciones informativas y un diseño sencillo orientado a la navegación del usuario.
🎯 Objetivo del proyecto
Aplicar los conocimientos básicos de:
Estructura HTML
Estilos con CSS
Organización de archivos en un proyecto web
